# title:  
Line bot -word counter   
# description:  
Line Message Apiを使った文字カウントのラインボットです。  
友達追加後、文字列を送信すると、空白を含む場合と含まない場合の文字数を返信します。
# Skills:  
バックエンド：Django  
その他：Line Message Api
